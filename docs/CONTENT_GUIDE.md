# CONTENT GUIDE

Короткая памятка для обновления контента на сайте SimanDev.

## Где редактировать тексты

Все тексты и контент лендинга находятся в файле:

- `frontend/lib/content.ts`

Там лежат:
- меню;
- заголовки и описания секций;
- карточки услуг и преимуществ;
- тарифы и CTA-кнопки;
- FAQ;
- контакты (email, telegram, бренд).

## Что обычно меняют

- **Текст секций**: блок `SECTIONS`
- **Цены и состав пакетов**: массив `PLANS`
- **Телефон/контакты/ссылки**: блок `CONTACTS`
- **FAQ**: массив `FAQ`

## После изменения контента

Из корня проекта:

```bash
cd frontend
pnpm typecheck
pnpm lint
pnpm build
```

Если всё зелёное — задеплойте обновлённую версию (через pipeline/деплой-скрипт).

Важно: это статический лендинг, поэтому изменения текста применяются после пересборки и деплоя.
